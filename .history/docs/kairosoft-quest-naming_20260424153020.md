## 命名パターンのテンプレ

カスタム家事の自動ネーミング用に、以下のパターンを辞書化して使うのもアリ。

```
[対象]の [敵/修飾]を [動作]せよ
例: [お皿]の[山]を[征服]せよ ／ [カビ]モンスターを[討伐]せよ

[場所] [効果形容]大作戦
例: 玄関ピカピカ大作戦 ／ 部屋すっきり大作戦

[対象] [オノマトペ]
例: お皿ぴかぴか ／ まど きらり
```

## ハイブリッド系（RPG×オペレーション）

RPGバトル系の語彙（モンスター／魔王／勇者／騎士／魔法陣）と、オペレーション系の
末尾語（作戦／任務／計画／大作戦）を組み合わせた、もっともゲーム感の強い統合案。
冒険の重みと作戦の達成感を両立する。

### ハイブリッド命名の公式

```
[RPGの敵/職業] + [討伐/召喚] + [作戦/任務/計画] = 王道ハイブリッド
例: 泡モンスター+討伐+作戦 → 泡モンスター討伐作戦

[場所/対象] + [Op動詞] + [大作戦]
例: 浴室+クリーン化+大作戦 → 浴室クリーン化大作戦

[RPG属性] + [魔法陣] + [発動作戦]
例: 光+魔法陣+発動作戦 → 光の魔法陣発動作戦
```

## 採用決定版（プランC: メイン名 + サブラベル）

知的障害のあるユーザーを含む精神年齢の多様性を考慮し、
ゲーム風メインタイトルの直下にひらがな中心の直接表現を併記する
「プランC（補助ラベル付き）」方式を採用する。

### 表示仕様

各クエストカードは2行構造で表示する。

```
[メインタイトル（ゲーム風）]
[サブラベル（ひらがな中心・直接表現）]

例:
🫧 泡モンスター 討伐作戦
   おふろ そうじ

🐕 相棒勇者 巡回作戦
   いぬの さんぽ
```

### 採用基準（今後のカスタム家事命名にも適用）

1. **メインタイトル** — ゲーム風（ハイブリッド系優先）
   - 冒険感・ワクワク・達成感を演出
   - 対象物／場所が明示されていると理解を助ける
2. **サブラベル** — 直接表現（ひらがな中心）
   - 知的障害のあるユーザー・低学年・認知負荷が高い状況でも即理解可能
   - メインタイトルと意味を冗長にすることでロバスト性を確保
3. **動作動詞の選好**
   - 「磨き上げ」「討伐」「収集」「浄化」など、視覚化しやすい動作語を優先
   - 「浄化」は動作が見えにくいため、「磨き上げ」で置換する判断を標準とする
