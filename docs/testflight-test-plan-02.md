# TestFlight テスト詳細 #2

> バージョン: v0.22.0 (Build 7)
> テスト期間: 2026-04-20〜
> 対象: iOS（TestFlight内部テスト）

---

## テスト対象バージョンの主な変更点（v0.15.0〜v0.22.0）

- Habitica風ピクセルアートSVG全面導入（38種+追加22種）
- ダンジョンテーマ（ダークパープル+ゴールド）
- SVGアニメーション全面導入（8種アイドルアニメ+歩行アニメ）
- ペットシステム（卵ドロップ→孵化→成長、6種×4段階）
- デイリーログインボーナス（7日サイクル、5〜50円）
- ショップ（称号8種購入）
- トロフィーケース（バッジ一覧）
- 家族管理画面改善（漢字化、レイアウト統一）
- WCAG AA準拠ダークモード視認性改善
- ピンチズーム全画面対応
- PIN入力改善（4桁自動ログイン）
- ウォレット3カードタップ遷移
