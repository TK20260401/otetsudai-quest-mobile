# おこづかいクエスト 使い方ガイド

**バージョン**: Phase 4 完了時点　**更新日**: 2026-04-14

---

## アプリ概要

お手伝い＝「クエスト」をクリアしてコインを稼ぐ、子ども向けマネー教育アプリです。  
稼いだコインは **つかう・ためる・ふやす** の3つのおさいふ（ポケット）で管理します。

---

## 画面構成

```
ランディング → ログイン → 子どもダッシュボード / 親ダッシュボード
                              ↓                        ↓
                        おさいふ詳細            クエスト管理・承認
                              ↓
                        つかうリクエスト
```

---

## 子ども向け操作

### 1. ログイン
- ランディング画面の **🔑 ログイン** をタップ
- 家族名を選択 → 自分の名前をタップ → PINコード（4桁）を入力

### 2. ダッシュボード
- **クエスト一覧**: 今日のお手伝いが表示される → タップで「できた！」報告
- **おさいふカード**: 3ポケットの残高を表示 → タップで詳細へ
- **リクエスト状況**: つかうリクエストの最新状態（⏳まち / ✅OK / ❌NG）

### 3. おさいふ詳細
- 合計残高と3ポケット（💰つかう / 🐷ためる / 🌱ふやす）の内訳
- **🛒 つかいたい！** ボタン → つかうリクエスト画面へ
- ちょきん目標の進捗バー表示
- 取引履歴（タブで絞り込み: ぜんぶ / 💰 / 🛒 / 🐷 / 📈）

### 4. つかうリクエスト
1. 金額を数字パッドで入力
2. 何に使いたいか入力（例: 「えんぴつ」）
3. 確認画面で内容チェック → **おねがいする！** で送信
4. 親が承認するまで ⏳ まち状態

---

## 保護者向け操作

### 1. ログイン
- メールアドレス + パスワードでログイン

### 2. 親ダッシュボード
- **クエスト管理**: クエストの作成・編集・削除、報酬コイン設定
- **子どもの完了報告**: 承認 / 却下（リアクション付き）
- **つかうリクエスト承認**: 子どもからのリクエストを確認 → 承認 / 却下
- **家族メンバー管理**: 子どもの追加・PIN設定

## テーマ（カラーパレット）

3種類から選択可能（全てCUD対応・色覚バリアフリー）:

| パレット | 特徴 |
|---------|------|
| 🌲 やさしい森 | 落ち着いた深緑＋クリーム。感覚過敏に配慮 |
| ⚔️ わくわく冒険 | インディゴ＋オレンジ。RPG風の達成感 |
| 🍃 そよかぜ | エメラルド基調。標準テーマ |

---

## アクセシビリティ対応

- **全漢字ルビ付き**: 未就学児〜低学年でも読める
- **大きなボタン**: 48dp以上のタッチターゲット
- **色＋アイコン**: 色だけに頼らない情報表示
- **カスタムアラート**: ルビ対応のモーダルダイアログ
